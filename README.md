Global Tourism Performance Dashboard

Project Overview
This interactive Power BI dashboard analyzes global tourism performance, focusing on revenue trends, traveler behaviors, and operational risks (sustainability and security).

Key Insights & Risk Findings
* Total Revenue: $115.50M with heavy reliance on third-party OTAs ($39.03M).
* Sustainability: Eco Adoption Rate stands at 21.38%, requiring better integration of green incentives.
* Operational Risks: 9.84% of travelers face severe language barriers, and safety perceptions sit at 39.26%.
